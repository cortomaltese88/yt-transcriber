# Security

`yt-transcriber` puo' trattare contenuti sensibili, inclusi audio, video e trascrizioni.

- Non caricare in issue pubbliche dati cliente, file audio/video riservati o trascrizioni reali.
- Segnalare vulnerabilita' o problemi privacy tramite un canale riservato verso il manutentore del progetto.
- Non inserire token, API key o altre credenziali nel repository, nelle issue o nei log condivisi.
- Verificare sempre output e file generati prima della condivisione o della pubblicazione.
